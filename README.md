Pycco is a Python port of Docco: the original quick-and-dirty, hundred-line-
long, literate-programming-style documentation generator. 
This is a modified version that uses dycco 1.0.2 
(from https://github.com/rojalator/dycco - pending acceptance into dycco itself) for Python files
